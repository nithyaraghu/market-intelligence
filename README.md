
# Market Intelligence — AI Stock Advisor

An AI-powered stock advisor with 5 specialist agents, live market data, and portfolio intelligence.

## Repositories
| Repo | Description |
|------|-------------|
| [Backend](https://github.com/nithyaraghu/AI-Stock-Advisor-main) | Flask + Python + PostgreSQL |
| [Frontend](https://github.com/nithyaraghu/AI-Stock-Advisor) | React + Vite dashboard |
| [Tests](https://github.com/nithyaraghu/market-intelligence-tests) | Playwright E2E test suite |

## Live Demo
[ai-stock-advisor-phi.vercel.app](https://ai-stock-advisor-phi.vercel.app)

## Tech Stack
Llama 3.3-70b · Groq · React · Flask · PostgreSQL · Finnhub · Polygon.io · NewsAPI · Playwright
